## 定义

`实体`：具有唯一标识的对象，可变的

`值对象`：没有唯一标识的对象，不可变。比如一单交易执行，它就不能改变状态了

`聚合`：由一组相关的实体和值对象组成的集合，它们之间是有关联的，它们之间的关联是有规律的，它们之间的关联是有方向的。

`聚合根`：聚合的入口，它是聚合的唯一标识，它是聚合
